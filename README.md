# Spotify Song Analysis

## 📊 Project Overview
This project analyzes Spotify song data to identify factors that influence a song’s popularity, measured by streams.

## 📁 Dataset
The dataset includes features such as:
- streams
- playlist counts
- bpm
- danceability
- mode (major/minor)
- artist count

## 🔍 Methods
- Data cleaning (converted streams to numeric)
- Permutation test (mode vs streams)
- Correlation analysis (streams vs playlists, bpm)
- Regression line and residual analysis
- Bootstrap confidence interval
- Grouped analysis (danceability, bpm, artist count)

## 📈 Key Findings
- Playlist exposure strongly correlates with streaming success  
- No statistically significant difference between major and minor mode songs  
- BPM shows no clear relationship with streams  
- Danceability does not show a strong pattern  
- Artist count does not have a consistent effect

- numpy
- matplotlib
- Jupyter Notebook
